Instructions
============
  TScaleLine is tested with Delphi 6. Please report it when you have another
  version of Delphi that works with the TScaleLine component.

  Installation:
    - Start Delphi
    - Click the menu Component, Install Component
    - Point to the ScaleLine.pas file
    - Click OK and recompile your user package

Use
===
  All properties are available through the Object Inspector.
